## Resumo

Sou estudante de Engenharia de Software na Universidade Federal do Amazonas (UFAM), com interesse em soluções que unem software, eletrônica, dados e inteligência artificial.

Tenho experiência acadêmica e prática com prototipação de sistemas embarcados, aplicações web/mobile, análise de dados e projetos de IoT voltados para monitoramento, automação e tomada de decisão.

Meu foco é construir soluções funcionais, bem estruturadas e aplicáveis a problemas reais, explorando desde microcontroladores e sensores até interfaces digitais e modelos de IA.
